    Online Materials HB

    The scryptIQ course provides educational materials for five modules: Python Fundamentals, Data Processing, Networks, Classical Machine Learning, and Artificial Intelligence.

    Each module comes with a stand alone online material portal which can be accessed via a link in each lesson repository. The online portal homepage will contain links to the module resources, such as:

    • this handbook

    • the course calendar

    • 1-to-1 support

    • the Python Orientation material

    Additionally, as each lesson is released they will appear in the right hand column of the port. There will be four lessons for each module (with the exception of Networks)

    Navigating the webpages

    Within every lesson release repository there will be a link to the introduction page for this lessons learning materials. Clicking homepage or the scryptIQ logo in the top left will take you to the homepage. Clicking the scryptIQ button in the top right will take you through to the your cohort section on the courses website.

    REMEMBER

    If the latest lesson releases aren't showing in your homepage after refreshing the page, clear your browser cache and refresh again.

    Downloading

    Every lesson, including the Orientation and Handbook, can be downloaded for you to keep. We will give you access to the web hosted materials for 2 years from when you first start the course, however after this time we will remove the GitHub organisation hosting the content.

    If you wish to keep the material beyond this time point you can download the online materials as both a PDF, which will be static, and as a Jupyter Notebook. With the notebook you can run the code, recreating the content you see in the web browser. This is also recommended for when you want to complete the practice exercises given in the materials locally, rather than in the browser (see below). Writing the answers in your own copy means you can save the work as you go, adding complimentary notes to the materials for future revision.

    How to download the materials:

    In the sidebar, click on resources. Doing so will reveal a sub-menu containing links to the Glossary, Downloads, and Handbook. Click on Downloads.

    Once you have clicked on Downloads you will be taken to a new page which will look like below. Clicking either Download All PDFs or Download All Notebooks will download either the pdfs or notebooks respectively. The files will be in a zipped folder that will unzip when double clicked.

    Pyodide

    The scryptIQ written materials are written in our new parchmentIQ template, which runs off a tool called Pyodide. Pyodide is a very useful software package that allows us to run live Python code cells from inside the webpage itself; thus, the code you see inside the materials has actually run in real-time. Pyodide also allows us to generate editable code cells, in which you can edit the Python code, and immediately try out what you've learned, without having to leave the web page.

    Types of Cells

    Code cells that have a green output box below are fixed and can not be user-edited.

    However, those with a pink output box below and a pink Edit button, can have their code altered by the user and can be run independently, each time you hit the Run Code button. Data stored in variables from previously run cells, both fixed and editable, should carry forward into subsequent editable cells.

    IMPORTANT

    If you refresh the webpage in your browser, this will restart and erase Pyodide's memory, which will overwrite any previously executed actions, restoring the page to its previous state.

    There will also be Practice Exercises scattered throughout the materials. These questions should be quick problems to solve, focusing on the subject material you are reading. Answering these earnestly should help progress your understanding much more quickly.

    Pyodide Issues

    Rarely, a code cell may not render with Pyodide. To rectify this, just refresh the page.

    If this or any other issues occur, please contact us at hello@scryptiq.ai or report the problem via our course feedback form.
